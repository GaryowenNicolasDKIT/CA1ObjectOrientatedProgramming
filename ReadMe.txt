This programs is designed to be able to provide websites with the database structure necessary to be able to keep a record of each user on the site. Such information includes the Users name, what game they are doing speedruns on(beating a game as fast as possible), how many times they posted a run (a recording of them beating the game as fast as possible) of that game to the site, their community rating (how liked they are within the community), whether or not they have a world record run (fastest time to beat a category), when they joined the site and the most recent run they have posted to the site.
